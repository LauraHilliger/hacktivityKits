What you'll learn

This tutorial will lead you through the necessary steps to customize, create and curate your own Maker Party event report for the web by achieving the following skills outlined on the Web Literacy Map's "Building" strand:

  • Inserting hyperlinks into your webpage
  • Combining and embedding multimedia content
  • Identifying and using HTML tags
  • Structuring and publishing your own webpage
  • Using CSS tags to change your webpage's style and layout

Let's get started!

Click on the blue arrow above to move to the next page and start your adventure like these young learners at a Maker Party in Bangalore last year!

First, let's change page titles...

Meta name: Let's start by changing the name of your webpage on line 5. Because this title sits within your page's 'meta' tags, it is a part of the metadata which ensures your page has its own identity across the web.

Title and subtitle: While you're at it, you'll also want to change the title and subtitle of your event report on the webpage itself. Head out of the page's metadata section to its CSS styling section on line 65 and line 69, and make those your own, too.

And insert a link to your Net Neutrality Maker Party event page.

Link it up: A big feature of Maker Party is the amazing events database which highlights Maker Parties from all over the world, big and small. Don't forget to create a page for your event before it happens, to help with promotion and attendance! Already finished? Link it on line 117 after the 'br' tag.

Bonus!

Want to learn more about the metadata on your page?

Try clicking on the other 'meta' element visible on line 6. If you click on the yellow question mark that appears, it'll take you to the Mozilla Developer Network, which keeps a set of openly-licensed documents to explain the many elements of the web in detail.

Let's give your page a header + background.

Logo: If you go to line 113, you'll notice another image URL under the 'header' element. This is where you can place your own baner or header image. We've included this Maker Party banner to get you started.

Background: On line 12 you'll also see the URL which allows the webpage to display your own event photo for the background. We've chosen this image from the Maker Party Resources page, but feel free to replace it with an image from your event!

Next, let's customize the rest!

Check out line 110. This is where CSS styling meets your own HTML content to make it all come alive. We'll leave you here to do some of your own learning by making, so go ahead and try putting in your own event agenda, slideshow, images, and text. Don't worry to mess things up and play around; just hit 'Save' first!

Bonus!

We've put some helpful headings in black, bold text as 'code comments' for help figuring out which section is which. And here is a post about funny code comments for those who want a bit of a laugh.

Last steps, and then publish!

Once you've customized to your liking, decide how you'd like your content to be attributed (here are details on Creative Commons licenses) and hit 'Save' one last time so your creation is published for the world to view and remix!

There, you did it. We're all proud of you! Don't forget to tag your creation with "makerparty" so it shows up in the global Maker Party Quilt for all to see!
