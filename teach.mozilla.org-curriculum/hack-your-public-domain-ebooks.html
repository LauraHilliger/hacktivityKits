Hack your (public domain) ebooks make activities

Add the Web to Anything

Even ebooks

Draw the Big Picture

I want my participants (people who teach others) to learn that:

  • ebooks are just mini-websites that can be easily edited.
  • HTML can be used to add content and CSS can be used to style existing or new content.
  • There are lots of public domain resources on the web that can be freely augmented
  • There are lots resources on the web for enhancing learning.
  • You are only limited by your imagination.

By the End of this Project

My learners will customize their own (or public domain) ebooks using basic HTML and CSS tags.

We're going to use a variety of tools.

Activate Essential Questions

My learners will be able to discuss and reflect upon:

  • What is the web made out of?
  • How learning a little HTML & CSS encourages creativity?
  • What does "open web" mean?

With a little knowledge of HTML, CSS you can augment your own public domain ebook.

Summarize:

By hacking ebooks my participants will learn about the open web, HTML and CSS. They will have a better understanding of what resources are available to them and how they can be used creatively.
