Activity

Setting Learning Objectives

Use this fun activity to collectively brainstorm learning objectives to build content for.

Made by Laura Hilliger

Steps for the Activity

  1. Break into Groups. No one has to work alone! Try to keep your groups to 5 people or less, so that each participant has a chance to share their thoughts. Give each group a stack of post-it notes and a marker or pen.

  2. [15 mins] Define the Learning Objectives

    Instruct learners that they should write at least 1 post it per question, per person, but that they should discuss the exercise so that the responses benefit everyone. Tell them to use single words or brief phrases, no long sentences!

    • What is the main topic you want people to learn about at your event?
    • What web skills do you want people to learn at your event?
    • What other skills do you want people to learn at your event?

  3. [15 mins] Organize what's on the wall

    Have everyone post their post it notes on a wall. Next, have participants read each others post-it notes and categorize them into "Topics", "Web Skills", "Other Skills". Any synonymous post-its should be placed together. Facilitate the organization procedure so that everyone gets involved. Have participants move post-its into columns underneath the category label. It should look like a mosh pit of collaboration.


    Photo by Pierros Papadeas

  4. Return to Groups

    Give each group at least 5 post its from each category. Don't worry about giving people their own post-its!

  5. [10 mins] Align with the Web Literacy Map

    Ask participants to compare their learning objectives with the Web Literacy Map. Which Web Literacy Competencies does each group's post-its cover? Have participants mark which post it aligns with which competencies.

  6. [10 mins] Have each group present quickly (you might even want to use a timer). Encourage peer feedback.

Materials

  • Markers or pens
  • Post-its
  • Paper

Discussion

Use the organization procedure to discuss overarching themes. Have participants talk as they organize, have them ask for clarification, modify post its, etc.
