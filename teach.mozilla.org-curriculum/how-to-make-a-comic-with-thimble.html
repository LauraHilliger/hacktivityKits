How to make a comic with Thimble

This is a quick and easy introduction to HTML and CSS as you make your own comic using Thimble.

Description

In this project you can create your own comic strip using and square photos you have - so why not use the photos from your Instagram account? You'll be able to add images, add copy in speech bubbles and captions. You'll also learn how to make more scenes by adding more comic boxes.

Learning Objectives

What specific skills will participants learn? What new knowledge will they come away with? You might want to list these individually, like this:

  • Changing images
  • Changing text
  • Changing classes
  • Copying nodes

Change some headers

First why not change the header <h1> to give your comic a better name. Any text that goes inbetween the <h1></h1> tags will become the heading.

Understand the HTML structure

The <div> with the class 'comic' wraps around each comic 'scene' to contain them all.

Swap images

Start by changing the images in the scene to one of your own photos. As the scene is square a square photo will work best. Why not get the image url of one of your Instagram photos?

Add Captions

Look at the 1st scene. It has both a caption and a speech bubble. By copying the speech paragraph <p>, try adding a speech bubble to the 2nd scene below and then change its text.

Reassign classes

Depending on the photo you use you might want to move the captions to either the 'top' or 'bottom', or the speech bubble either 'left or 'right'.

Look at the 4th scene. It has a speech bubble on the left side. Change the class name 'left' to 'right' to make the speech bubble move to the right side.

If you want a caption to be at the top make sure it has the class 'top', if you want it at the bottom makes sure it has the class 'bottom'.

Want to create a larger comic?

Copy the HTML for an entire 'scene' and paste it below the last scene...you now have 3 more scenes in your comic ;)

Congrats! You just made a comic!

Did you realize you were also learning a few things in the process? You actually learned a bunch about HTML structures. Plus, you know how to wrangle images in HTML as well.
