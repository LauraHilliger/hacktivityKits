Activity

Brainstorming, Outlining and Curating

This activity helps participants use predefined learning objectives to make a plan for a Teaching Kit and supporting materials.

Made by Laura Hilliger

Steps for the Activity

  1. [5 mins] Break into Groups. No one has to work alone! Try to keep your groups to 5 people or less, so that each participant has a chance to share their thoughts. Give each group paper, markers and pens. Make sure each group has at least one computer as well.

  2. Review the Learning Objectives

    Instruct learners to review the learning objectives they would live to create content for. If participants haven't set learning objectives, you can run this activity or choose skills and competencies from the Web Literacy Map.

  3. [30 mins] Brainstorming

    Have learners brainstorm ways they might teach each learning objective. Bonus points for using a single method to teach multiple objectives. Have learners document their thinking using the Teaching Kit Design Canvas or using collaborative mindmapping or another technique.

  4. [20 mins] Outlining

    Have each group outline everything they'll need to create digitally documented activities that anyone can run. Have them think about the images, videos and text they'll need to explain how to teach each learning objective.

  5. [25 mins] Curating

    Introduce the concept of copyleft and open content, and have groups collect openly licensed digital resources. Participants might find suitable pieces and parts at Webmaker.org, DigitalIs or P2PU.org. Have groups collect resources and add URLs to an Etherpad.

  6. [10 mins] Have each group present their plans quickly. Encourage peer feedback. And if you have time, get people building their plans!

Materials

  • Computer
  • Web Access
  • Markers or pens
  • Post-its
  • Paper

Discussion

Use the presentation time to reinforce the idea of copyleft. Who else has written a open educational resource designed to teach specific topics or learning objectives? How can one reuse what other's have made? How does one find resources?
