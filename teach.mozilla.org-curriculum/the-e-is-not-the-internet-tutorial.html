The "E" is not the Internet

This make is designed for people who are very novice web users. It's designed to be done with a more advanced user. You can use this make to explain a little bit about the history of web browsers and how to navigate.

Description:

Lots of people click on the Internet Explorer icon thinking that that IS the Internet. This make is designed to help someone understand what a browser is, and learn a little bit of HTML and CSS in the process. Together, you'll make a new webpage!

Learning Objectives

  • Accessing the web using the common features of web browsers
  • Using hyperlinks to access a range of resources on the web
  • Understanding the difference between the Internet, the Web and a Browser
  • What HTML is and how to remix content

Give your page a witty title

Start by changing the title element on line 8 to a witty one for your webpage. Typically, the title will show up at the top of your browser window and will be searchable in search engines like Google. In Thimble, the title shows up in the tab above the pane on the right side of the editor.

Change the Browser icon

Now change out the image on line 10 with a different browser icon. Images are defined in HTML with the img tag. img is one of the few weird tags that you don't have to close. Every img tag has a src attribute where you include the URL to the image you want to use. To change the image on line 10, replace the content within the quotes with a url of another image.

If you are looking for images a good place to start is http://search.creativecommons.org/. Here you can find images that are available under the Creative Commons license. To get the URL of an image, right click or CTRL-click and select "Copy Image Address", or within Flickr, select Share, the copy the image URL.

Show what you know

Line 11 is styled with an h1 tag, where "h" stands for header and "1" means most important, followed by h2, h3, etc. It is used to build an outline or hierarchy into your page. Make a NEW header about the browser image your chose.

Line 12 is the subheader, use this to expand on your header statement

Change the look of your page

You've played with HTML, now try tinkering with the look of your page. To do so we will use CSS, which stands for Cascading Style Sheets,and lets you add a touch of style to your pages. All of the stuff within the style tags is CSS. You can look for the name of the corresponding HTML tag to change how it is styled. For example, to change the style of h1, look for "h1" below line 16.

Don't be afraid to play!

Try tinkering with the different kinds of style attributes. Maybe you want to change the formatting of the page by upping the size of the margins, or perhaps you are more interested in reducing the font size. Be playful and remember that things can easily be undone or redone by pressing the undo and redo buttons on the top of the editor.

Finally, let's fine tune at line 43. Change the position of the headings by adjusting the numbers in "padding-top" attribute. Here they are listed as px- which stands for pixels. Make incremental changes to see how the headings move. You can also change the colors, fonts, etc.

The end!

You've learned a little about the difference between the Internet, the Web and a browser AND you learned things like: how to set the title of a webpage (title tag), work with images (img tag), organize your text with headers (h1, h2, etc.) and start to get down and dirty in CSS to change the look and feel of your page. Good Webmaking my friend!
