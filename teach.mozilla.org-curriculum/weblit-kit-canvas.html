Activity

WebLit Kit Canvas

This activity asks learners to iterate on preexisting curriculum to strengthen alignment with the Web Literacy Map.

Made by Laura

Steps for the Activity

  1. [5 mins] Break into Groups. No one has to work alone! Try to keep your groups to 5 people or less, so that each participant has a chance to share their thoughts. Give each group paper, markers, pens and an activity or lesson from a preexisting piece of curriculum. Make sure each group has at least one computer as well.

  2. Review the Learning Objectives

    [10 mins] Instruct learners to review and briefly discuss the curriculum they've chosen.

  3. [30 mins] Brainstorming

    Have learners brainstorm ways they might teach each learning objective. How do their thoughts compare with the activity? How could the activity be more learner centric? Bonus points for using a single method to teach multiple objectives. Ask them to define any missing objectives. Have learners document their thinking on paper using collaborative mindmapping.

  4. [15 mins] Circle Up and Debrief

    Have everyone return to the circle and present their mindmaps. What did they notice about the Web Literacy Map as they referenced it in context with a piece of curriculum? Have each group talk about the improvements they made, where they got stuck and if the Web Literacy Map resonated with them.

Materials

  • Computer
  • Web Access
  • Markers or pens
  • Post-its
  • Paper

Discussion

Use the presentation time to reinforce the idea that you can bake the web into anything. Talk about using webmaking to teach specific topics or learning objectives.
