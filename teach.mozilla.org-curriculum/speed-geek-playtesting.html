Activity

Speed Geek Playtesting

Use this activity to quickly share things people have made with others at an event.

Made by Laura

Steps for the Activity

  1. Set up a table for each group of people working on a project. Have each group choose a person to "run" that table.

  2. Tell participants they will have 3 minutes to talk about a project or hear about a project.

  3. Once the runners have set up at their tables, divide the rest of the participants into groups and have groups wait at the various tables.

  4. When you're ready, say go and time the first three minutes. After the first speed geek round is finished, rotate the groups and start the timer again. Repeat this until each group has been to each table.

Discussion

Encourage participants to take notes and give feedback!
