So what are CSS positions?

Well a CSS position is a way of placing something on the screen in a specific place. We will be focusing on absolute positions. These are absolutely where we say they are. We have also fixed the size of the game to a set size (800px x 600px) to make life a little easier for you.

What's the difference between em and percents?

An em is a multiple of a predefined size. The predefined size is the current font size, so 1em equals the current font size. In this case, the font size is 20px, so an em is 20px. That means 2em is 40px, and so on.

You can also use percentages for positioning...the %, as you would expect, represents a percentage of the page. So the distance the item moves will depend on the size of the screen.

How do I use HTML comments?

The comment tags look like <!-- and -->. You can make comments in HTML to let people know things about your code. Hit the remix button at the top right of this tutorial and look for bold, black text surrounded by <!-- and --> in the code editor, those are HTML comments.
