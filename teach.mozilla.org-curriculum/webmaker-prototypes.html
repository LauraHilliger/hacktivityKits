WebLitMapper

The WebLitMapper is a bookmarklet that allows users to tag content around the web with the competencies that make up the Web Literacy Map. It's a similar idea to social bookmarking, but with a controlled set of tags.

Go

Kit Builder

The Kit Builder prototype aims to change the way we think about and make learning materials by making inherent remixability, modern assessment and web friendliness part of the curriculum itself.
