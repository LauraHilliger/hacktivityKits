Module Subtitle

Draw the Big Picture

For example: I want my participants to learn:

  • the Web is made out of building blocks that they can rearrange to create new things through an iterative process.
  • HTML is a language used to tell computers how to build webpages so that humans can consume them. CSS means cascading style sheets.
  • Parts of a webpage include HTML, CSS, graphic assets and text content
  • The combination of an opening tag and its corresponding closing tag and the content in between is called an element
  • There are resources on the web for webmaking
  • Hacking is FUN!

Think of the Results

For example: My learners will make their own webpages using basic HTML tags. They'll also style their webpages using the following CSS properties: positioning values, hex numbers/color, fonts/sizes, class, div

We're going to use Thimble!

Activate Essential Questions

For example: My learners will be able to discuss and reflect upon:

  • What is the web made out of?
  • How does HTML & CSS allow me to be creative?
  • How did feedback help me iterate and improve on my project?
  • What does "open web" mean?

Note: The purpose of this exercise is to create an outline of learning goals that you can design activities around.
