Decode the Code

Offline Hacktivity

Diving In

Learners work in groups with this sheet and the HTML cheatsheet to figure out what the code means. Then digital images are introduced and learners make their first image hack.

Back to the overview View other kits

Steps

  1. Ask if anyone is curious about what the code is actually telling the computer to do? Hand out HTML cheatsheets and Decode the Code. Ask learners to work in small groups to interpret the code. Give them crayons or markers so they can translate the HTML into words and images.

  2. Have each group show one of their interpretations and explain it (e.g., “We put this sentence in bold because right before it was <b> and right after it was </b>".) It may be easier for them to write the tags than to describe them.

  3. Introduce digital images to the group: Explain that if they see .jpg,.png or .gif it implies that they are looking at a digital image that has been uploaded to the web. You may choose to show them an example of this.

  4. Have learners move to their computers, then do the Hack Great Science for Girls: with the XRay Goggles Hacktivity

Materials

  • Decode the Code print out
  • Computer
  • Web Access
  • Projector
  • A blog or Etherpad for posting work and/or links
  • HTML Cheatsheet
  • CSS Cheatsheet
  • Goggles Cheatsheet

Discussion

At the end of the session, have participants present their new webpages, explain their work process and what they learned.
  • Go over the steps again.
  • Look at the code, point out the tags, attributes and elements.
  • Hack one part of the webpage. Hit the “commit changes” button to save changes locally on your computer.
  • Share: hit “P” to publish the new webpage on the Internet. Take a screenshot or copy and paste the code into a HTML document.
