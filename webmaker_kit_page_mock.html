X-Ray Goggles Kit

This kit's hacktivities and resources will help people learn basic concepts of HTML, CSS and the Open Web. Learners will tinker around with web pages using the X-Ray Goggles and remix to make their own webpages.

Thimble Kit

The Thimble hacktivities will help people learn to write HTML, CSS and make on the Open Web. Learners will tinker around with Thimble and create their own webpages.

Popcorn Kit

The hacktivities in the Popcorn Kit will help people learn the concept of procedural storytelling, how to use Popcorn Maker and nuances of web native film.

Thimble for Journalists Kit

This introduction will help journalists become familiar with code they may encounter in their news organization's content management system and serve as an introduction for continued study into coding and journalism.

Online Storytelling

The Online Storytelling hacktivities delve into Digital Storytelling using Popcorn Maker as a mechanism for creative expression through story on the Web.

Revolutions in Media

Through the hacktivities in this Kit, learners will gain an understanding on how media has influence over culture and society, and cultures and societies have influence over shaping media. They'll use Popcorn Maker to remix context and bring stories to life.

Ways of the Web

Like the Thimble Kit, these hacktivities will help people learn to write HTML, CSS and make on the Open Web. Learners will tinker around with Thimble and create their own webpages.

Make it! Share it!

This Kit contains hacktivities that will help people learn how to plan and make an interactive and remixable How-to that uses the Web as it's platform.
